# 快速开启网页端新项目的模板

1. **插件**
   * vue-router  
   * echarts
   * element-plus
   * axios
   * pinia
   * sass  
   * vueuse

2. **自动导入**  
   * vue
   * element-plus
   * vue-router

3. **文件说明**
   * assets-->全局资源
   * components-->公共组件，内置了封装的Echart，并全局注册
   * router-->路由
   * store-->全局变量
   * style-->全局样式
   * utils-->封装好的函数和变量
   * views-->页面
